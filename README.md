## üöÄ Getting Started

### Running JavaScript Files
All JavaScript files in this repository are standalone and can be run directly with Node.js:

```bash
node <filename>
```

**Example:**
```bash
node 01_basics/dataTypes.js
node 03_basics/arrowFunction.js
node 05_Iterations/06_reduce.js
```

### Opening HTML Files
HTML files and DOM projects should be opened in a browser:

```bash
# Open in default browser (Windows)
start 06_DOM/01_one.html

# Or use VS Code Live Server extension
```

## üí° Learning Path

**Recommended order of study:**

1. **Start with Basics** (`01_basics/`, `02_basics/`)
   - Understand data types, arrays, objects, and strings
   - Learn how memory works (stack vs heap)

2. **Functions and Scope** (`03_basics/`)
   - Master function declarations and arrow functions
   - Understand scope, hoisting, and execution context
   - Learn callback patterns

3. **Control Flow** (`04_control_flow/`)
   - Conditional logic with if/else and switch
   - Understand truthy/falsy values

4. **Iterations** (`05_Iterations/`)
   - Master loops and array iteration
   - Learn functional methods: map, filter, reduce

5. **DOM and Events** (`06_DOM/`, `08_events/`)
   - Understand the event loop
   - Practice DOM manipulation
   - Build interactive projects
